# JIRA Clippy

Copy jira issues to your clipboard as 'CLC-999: Description + URL'.

[Download](https://chrome.google.com/webstore/detail/jira-clippy/ngpeembgljjbbgeejlcbaddnnejjkhkj)

## Build

To build the extension use:

```
chmod +x build.sh
./build.sh
```
